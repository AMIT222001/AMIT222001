### Antier Solutions Pvt. Ltd. &nbsp;·&nbsp; Software Engineer Intern
`Aug 2024 – Present` &nbsp;·&nbsp; Mohali, Punjab, India

**Trillionipay — Notification Service & User Onboarding**
- Built a centralized notification service with **Apache Kafka** + **Firebase FCM**, processing **10,000+ events/day** for OTP emails and push notifications
- Developed **10+ RESTful APIs** for KYC validation, wallet creation, and email verification — reduced manual onboarding effort by **40%**
- Enforced API security with **RBAC** middleware across multiple user roles

**Future Wallet — Redis Performance Optimization**
- Replaced iterative `HGET`/`HSET` calls with batched `HMGET` and pipeline writes, cutting network round-trips from **N → 2** per operation
- Achieved **85% reduction in Redis I/O** by staging updates in-memory before bulk writes — latency dropped from **120 ms → 18 ms**

**Miracle Wallet — Blockchain RPC Optimization**
- Replaced per-transaction `eth_getTransactionReceipt` calls with a single `eth_getBlockReceipts` call per block
- Reduced block processing time from **4.5 s → 95 ms** — a **97% improvement** — via local ABI decoding with ethers.js v6
